# Daily Journal

A responsive and interactive daily journaling web application with goal tracking features.

## Features

- Daily journal entries with multiple sections
- Weekly and monthly goal tracking
- Progress tracking with visual indicators
- Responsive design for all devices
- Local storage for data persistence
- Clean and intuitive user interface

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MohsinFurkh/Daily-Journal.git
   cd Daily-Journal
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

1. Start the server:
   ```bash
   node server.js
   ```

2. Open your browser and visit:
   ```
   http://localhost:3000
   ```

## Usage

- **Journal Entries**: Add and save your daily journal entries.
- **Weekly Goals**: Set and track goals for the current week.
- **Monthly Goals**: Set and track goals for the current month.
- **Progress Tracking**: Visual indicators show your journaling streak and goal completion.

## Technologies Used

- Node.js
- Express.js
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
